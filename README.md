# Leitbild Wiki

This repository contains the LLM-readable wiki and operational knowledge base for Leitbild, a map-based control-center research platform.

Rendered site: https://samsinn-wikis.github.io/leitbild/

The wiki is designed for both humans and AI agents. Humans should be able to understand what Leitbild does and try the demos. AI agents should be able to fetch a small number of Markdown files and recover the operating rules, scenario structure, domain semantics, and API expectations needed to monitor or act in a Leitbild simulation.

Canonical executable code and current scenario JSON live in [`michaelhil/leitbild`](https://github.com/michaelhil/leitbild). Scenario JSON in this wiki is mirrored for reading, review, and future agent retrieval.
